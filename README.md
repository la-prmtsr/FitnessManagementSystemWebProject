# FitnessManagementSystemWebProject
ASP.NET Core MVC kullanılarak geliştirilen kapsamlı bir Spor Salonu (Fitness Center) Yönetim ve Randevu Sistemidir. Sistem; spor salonlarının sunduğu hizmetlerin, antrenör uzmanlıklarının, üyelik bilgilerinin ve randevu süreçlerinin dijital ortamda yönetilmesini sağlar.

Uygulama; salon ve hizmet tanımlamaları, antrenör yönetimi, üyelerin uygun saatlere göre randevu oluşturabilmesi, randevu onay mekanizması, yapay zekâ destekli kişisel egzersiz/diyet önerileri ve LINQ tabanlı REST API servisleri gibi bileşenleri içerir. Ayrıca kullanıcı kayıt sistemi, rol bazlı yetkilendirme, veri doğrulama, CRUD işlemleri ve modern bir Bootstrap arayüzüyle profesyonel bir yapıya sahiptir.
